Apache Drill Overview

  • Low-latency 1 SQL query engine on Hadoop and NoSQL.
  • Part of the MapR offering:

Apache Drill Overview: Key Features

  • Generic: Drill supports multitude of data types and sources through a single interface: ANSI SQL.
  • Flexible: It can query nested data formats (JSON, Parquet) just like SQL columns.
  • Schema-less: There is no need to provide a schema or type specification for data before starting queries, Drill discovers these automagically.
  • Extensible: Different data sources, such as e.g.MapR-FS, MapR-DB, Hive, JSON etc. are handled through various different storage plugins.
  • Metadata: Drill uses a de-centralized metadata repository. Also, metadata is derived from storage plugins.
